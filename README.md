## 推导过程

根据图1中的直角三角形∆OBC，我们可以推导出以下转换公式：

1. **极坐标系到拟直角坐标系的转换**：
   - X = r * sin(θ) * cos(δ)
   - Y = r * sin(θ) * sin(δ)
   - H = r * cos(θ) + h

2. **拟直角坐标系到极坐标系的转换**：
   - r = sqrt(X^2 + Y^2 + (H - h)^2)
   - θ = arccos((H - h) / r)
   - δ = arctan(Y / X)
